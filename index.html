História dos computadores

(voltar)

A história dos computadores começou com ferramentas simples como o ábaco e evoluiu para máquinas mecânicas. No século XX, surgiram os computadores eletrônicos, são dividadas em 5 geração são elas :

  • 1ª Geração Usavam válvulas eletrônicas e ocupavam grandes espaços.
  • 2ª Geração Com transistores, ficaram menores e mais rápidos.
  • 3ª Geração Circuitos integrados trouxeram mais eficiência.
  • 4ª Geração Microprocessadores permitiram o surgimento de computadores pessoais.
  • 5ª Geração Foco em inteligência artificial, dispositivos móveis e computação em rede.

O que são Clients?

(voltar)

é um cliente que solicita páginas de um servidor web. exemplo Um navegador da web (como o Google Chrome).

Características:

  1. Solicita informações ou serviços.
  2. Depende de um servidor para processar e retornar dados.
  3. Exemplos de clientes: aplicativos web, clientes de e-mail, jogos online.

O que são Servers?

(voltar)

Servers (servidores) são dispositivos ou softwares que fornecem serviços, dados ou recursos a outros dispositivos ou softwares chamados clients (clientes), em uma arquitetura cliente-servidor.

Resumo de Funções:

  • Armazenar e processar dados: Servem informações solicitadas pelos clientes.
  • Gerenciar recursos: Como arquivos, bancos de dados, ou serviços (exemplo: e-mail, hospedagem de sites).
  • Responder requisições: Processam pedidos e enviam respostas aos clientes conectados

Exemplos:

  1. Servidor Web: Hospeda sites e envia páginas ao navegador.
  2. Servidor de Arquivos: Armazena e compartilha documentos em uma rede.
  3. Servidor de Banco de Dados: Gerencia o acesso a bancos de dados para aplicativos.

Como é uma estrutura básica de um HTML?

(voltar)

A estrutura básica de um documento HTML segue um formato padrão, que inclui elementos essenciais para criar uma página web.

Explicação dos Elementos:

  1. <!DOCTYPE html>: Declaração que informa ao navegador que o documento usa HTML5.
  2. <html>: Elemento raiz que contém todo o conteúdo do documento HTML.
    • lang="pt-BR": Define o idioma da página como português do Brasil.
  3. <head>: Contém informações meta sobre o documento (não visíveis na página).
    • <"meta charset="UTF-8"< Define a codificação de caracteres para suportar acentos e símbolos.
    • <"meta name="viewport" content="width=device-width, initial-scale=1.0""< Ajusta a página para dispositivos móveis.
    • <"title""< Define o título exibido na aba do navegador.
  4. "body": Contém o conteúdo visível da página (como texto, imagens e links).

Essa estrutura é o ponto de partida para qualquer projeto web!

Entendendo a diferença entre listas ordenadas e não ordenadas

(voltar)

As listas ordenadas e não ordenadas em HTML são usadas para organizar itens de forma diferente, e a principal diferença está no formato de exibição dos itens.

1. Lista Ordenada <(ol)<

  • Uso: Exibe os itens em uma sequência numerada ou com algum tipo de ordenação.
  • Elementos: A lista ordenada é criada com a tag <(ol)<, e cada item da lista é marcado com a tag <li< .
  • Aparência: Os itens são numerados automaticamente. Você pode personalizar o estilo da numeração com CSS (por exemplo, números romanos, letras, etc.).

1. Lista Não Ordenada <(ul)<

  • Uso: Exibe os itens sem uma ordem específica, geralmente com marcadores (como círculos ou quadrados).
  • Elementos: A lista ordenada é criada com a tag <(ul)<, e cada item da lista é marcado com a tag <li< .
  • Aparência: Os itens são exibidos com marcadores (padrão: círculos). Você pode mudar o tipo de marcador com CSS (por exemplo, quadrados, imagens personalizadas).
